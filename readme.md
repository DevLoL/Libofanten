# Libofanten

## Reference

**begin**
Initalize Library. Required parameter is the number of panels.

Usage:
`Blof.begin(panelcount)`

**clear**
Clears all pixel settings to off.

Usage:
`Blof.clear()`

**setBlink**
Usage:
`Blof.setBlink(col, val)`

**setPixel**
Set a pixel

Usage:
`Blof.setPixel(col, row, val)`

**off**
Deactivate Flushing

Usage:
`Blof.off()`

**on**
Activate Flushing

Usage:
`Blof.on()`

**flush**
Flushes pixel settings to panels.

Usage:
`Blof.flush()`
